SSH框架学习（Spring + Struts2 + Hibernate）

📌 一、什么是SSH框架？

SSH 是 Java Web 开发中的经典组合：

  • Spring：负责管理对象（IOC）和事务
  • Struts2：负责 MVC 控制层
  • Hibernate：负责数据库持久化（ORM）

🧱 二、整体架构

Spring 管理所有对象

Struts2 交给 Spring 管理

Hibernate 由 Spring 管理事务

1️⃣ Spring 配置（applicationContext.xml）

2️⃣ Struts2 配置（struts.xml）

3️⃣ Action 类（控制层）

4️⃣ Hibernate 实体类

5️⃣ DAO 层（Hibernate操作）

6️⃣ Service 层（Spring管理）
